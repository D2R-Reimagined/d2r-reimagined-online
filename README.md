# D2R Reimagined Online

D2R Reimagined Online is an unofficial D2RLAN-compatible build of **D2R Reimagined**, intended for single-player and TCP/IP multiplayer through a compatible D2RLAN setup.

This project is maintained separately from the official D2R Reimagined mod.

Official D2R Reimagined repository:  
https://github.com/D2R-Reimagined/d2r-reimagined-mod

## Status

This is **not an official version** of D2R Reimagined.

Bugs, crashes, missing features, multiplayer issues, or compatibility problems in this build should be treated as issues with this D2RLAN compatibility version, not with the official D2R Reimagined project.

The current **3.x version** is the recommended stable version of this build.

## Purpose

This version exists because D2RLAN uses an older D2R client environment than the latest official D2R client.

It is designed for players who want to play D2R Reimagined in:

- Single-player
- TCP/IP multiplayer
- A D2RLAN-compatible setup

Because this build targets an older client environment, some features from the latest official offline version may be unavailable or behave differently.

## Supported Features

- TCP/IP multiplayer through D2RLAN
- Most D2R Reimagined gameplay systems
- Most classes in Reimagined, excluding Warlock
- Reduced risk of breaking from official D2R client every updates
- Some hardcoded modding possibilities not easily available in the latest client
- many shared stash tabs

## Known Limitations

### Warlock

Warlock class is not available in D2RLAN. (However, it may be added later depending on D2RLAN support)

### Mosaic Assassin

Mosaic Assassin charge preservation is not currently supported.

### Stackable Stash

The stackable stash system is not currently available.

### Version Differences

Includes most updates from the official Reimagined version. However, some features are restricted or modified. Please refer to the docs.

## Multiplayer Notes

For TCP/IP multiplayer, all players should use:

- The same mod version
- The same D2RLAN-compatible client setup
- Matching mod files

Mismatched files may cause crashes, missing visuals, broken skills, or other unexpected behavior.
